# Step 2: Create a Simulation World Template<a name="worlds-creating-getting-started-template-gen"></a>

In this section, you create a simulation world template\. A simulation world template contains parameters that Simulation WorldForge uses to generate random worlds\. Simulation WorldForge provides sample templates to help you get started\. 

**To create a simulation world template**

1. Sign in to the AWS RoboMaker console at [https://console\.aws\.amazon\.com/robomaker/](https://console.aws.amazon.com/robomaker/)\.

1. On the AWS RoboMaker console, expand **Simulation WorldForge** on the left and then choose **World templates**\.

1. On the **World templates** page, choose **Create template**\.

1. On the **Create a world template** page, under **Sample templates**, choose **Browse and select**\.

1. On the **Templates** page, under **Medium house**, choose **Start**\.

1. On the **Template detail** page, choose **Save and exit** to save the simulation world template\. You will use it in the next section to generate a simulation world\.
